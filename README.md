# Sofía — Neon Portfolio Template
Portfolio template for WEB ATELIER (UDIT).  
Tech: **Bootstrap 5 (CDN)** + **Tailwind (CLI, prefixed `tw-`)** + **GSAP + ScrollTrigger + TextPlugin**.
